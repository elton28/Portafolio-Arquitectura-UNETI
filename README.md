# Portafolio-Arquitectura-UNETI

Portafolio digital para la materia de Arquitectura del Computador, Laboratorio VAC-SD1.

## Cómo ejecutar localmente

**Prerrequisitos:** Node.js

1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Ejecutar la aplicación:
   ```bash
   npm run dev
   ```

## Tecnologías utilizadas

- React 19
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
